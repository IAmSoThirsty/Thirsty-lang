# Thirsty-lang Dependencies

## Core Dependencies

### Node.js (Primary Runtime)
- **Version Required**: 14.0.0 or higher
- **Recommended**: 20.x LTS
- **Purpose**: Primary interpreter runtime
- **License**: MIT

### Python (Alternative Runtime)
- **Version Required**: 3.8 or higher
- **Recommended**: 3.10+
- **Purpose**: Alternative interpreter implementation
- **License**: PSF

## Node.js Dependencies (package.json)

### Production Dependencies
None - Thirsty-lang core uses only Node.js standard library

### Development Dependencies
None - Testing and tools use Node.js standard library

- Visual Studio Code 1.60.0 or higher

## Build Dependencies
None - Thirsty-lang is an interpreted language

## Runtime Dependencies
None - All interpreters use standard library only

## Notes
- Thirsty-lang is designed to have zero external dependencies for core functionality
- All features work with just Node.js or Python standard libraries
- Optional dependencies enhance but are not required for basic operation
- Docker is optional and only needed for containerized deployment

## Dependency Management

### Node.js
```bash
npm install          # Install dependencies (if any added)
npm ci              # Clean install
```

### Python
```bash
pip install -r requirements.txt           # Core dependencies
pip install -r requirements-dev.txt       # Development dependencies
./setup_venv.sh                          # Automated setup
```

### Docker
```bash
docker-compose build                     # Build images
docker-compose up                        # Start services
```

## Security
- Keep Node.js and Python updated to latest stable versions
- Regularly update dependencies for security patches
- Run `npm audit` for Node.js security check
- Run `safety check` for Python security check

## Version Pinning
- Development dependencies are pinned to major versions
- Core remains dependency-free for maximum compatibility
- Docker images use specific base image versions
