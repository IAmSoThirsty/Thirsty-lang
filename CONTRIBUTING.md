## Code of Conduct

Be respectful, inclusive, and keep the community hydrated and happy!

## How to Contribute

### Reporting Issues

- Check if the issue already exists
- Provide a clear description and example code
- Include the version of Thirsty-lang you're using

### Submitting Pull Requests

1. Fork the repository
2. Create a new branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Run tests (`npm test`)
5. Commit your changes (`git commit -m 'Add amazing feature'`)
6. Push to the branch (`git push origin feature/amazing-feature`)
7. Open a Pull Request

### Coding Standards

- Use clear, descriptive variable names
- Comment complex logic
- Follow the existing code style
- Add tests for new features
- Update documentation as needed

### Development Setup

```bash
git clone https://github.com/IAmSoThirsty/Thirsty-lang.git
cd Thirsty-lang
npm install
npm test
```

### Running Examples

```bash
npm start examples/hello.thirsty
```

## Language Extensions

We welcome contributions to all language variants:
- **Thirsty-lang**: The base language
- **Thirsty+**: Enhanced features
- **Thirsty++**: Advanced features
- **ThirstOfGods**: Ultimate power features

## Testing

All contributions should include appropriate tests. Run the test suite with:

```bash
npm test
```
