Four Tiers of Thirst

Start simple, grow powerful. Each tier builds on the last — fully backward compatible.

Tier 1

Thirsty-Lang

.thirsty

The foundation. Variables, output, and comments.

  • drink — declare variables
  • pour — print output
  • // — comments
Tier 2

Thirst of Gods

.tog

Logic and flow. Conditions, arithmetic, comparisons.

  • thirsty / hydrated — if/else
  • parched / quenched — true/false
  • Arithmetic & comparisons
Tier 3

T.A.R.L.

.tarl

Power tools. Functions, arrays, loops, security.

  • glass — function definitions
  • refill — while loops
  • reservoir — arrays
  • shield / armor — security
Tier 4

Thirsty's Shadow

.shadow

Enterprise. Classes, OOP, and shadow computing.

  • fountain — class declarations
  • this — instance scope
  • cascade — async (planned)
  • morph — obfuscation (planned)
