# Changelog

All notable changes to the Thirsty-Lang VS Code extension.

## [1.0.0] - 2026-02-23

### Added

- Full syntax highlighting for all Thirsty-Lang keywords
- Support for all four language family tiers:
  - Thirsty-Lang (`.thirsty`)
  - Thirst of Gods (`.tog`)
  - T.A.R.L. (`.tarl`)
  - Thirsty's Shadow (`.shadow`)
- Code snippets for common patterns (drink, pour, thirsty/hydrated, refill, glass, shield)
- Language configuration (auto-closing brackets, comment toggling, indentation)
- TextMate grammar with scopes for keywords, strings, numbers, operators, and comments
