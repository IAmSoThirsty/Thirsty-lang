# Changelog

All notable changes to Thirsty-lang will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-28

### Added
- Initial release of Thirsty-lang programming language
- Core interpreter with water-themed syntax
- Variable declarations using `drink`
- Output statements using `pour`
- Input statements using `sip`
- Interactive REPL (Read-Eval-Pour-Loop)
- Interactive training program for all skill levels
- Full-featured debugger with breakpoints and stepping
- Code formatter for consistent styling
- Linter for code quality checks
- Performance profiler for optimization
- Documentation generator (HTML and Markdown)
- AST (Abstract Syntax Tree) generator
- Multi-language transpiler (JavaScript, Python, Go, Rust, Java, C)
- Package manager for dependency management
- Web-based playground for browser execution
- VS Code extension with syntax highlighting
- Comprehensive documentation
- Example programs and tutorials
- GitHub Actions CI/CD pipeline
- Test suite for core functionality
- Benchmark suite for performance testing
